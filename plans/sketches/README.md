# Product Brainstorm Sketches

This directory contains product brainstorm outputs from the `talk` skill.

## Purpose

These are rough visual explorations of product ideas - think "whiteboard sketches" or "napkin drawings." They capture:
- User problems and context
- Visual concepts and approaches
- Chosen direction with key flows
- Open questions for PRD phase

## Workflow

1. **talk** skill → Creates brainstorm sketch here
2. **write-to-prd** skill → Consumes sketch to create formal PRD
3. **medium-plan** skill → Creates technical implementation plan from PRD

## File Format

Files follow the pattern: `YYYY-MM-DD-<topic>-brainstorm.md`

Example: `2026-02-21-user-dashboard-brainstorm.md`

## Note

These are exploratory documents that may change rapidly. They are NOT committed to git by default.
